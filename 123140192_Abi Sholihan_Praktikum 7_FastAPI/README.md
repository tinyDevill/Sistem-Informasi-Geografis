# Cara Menjalankan API:
1. Unduh atau Fork project ini
2. Buka folder project di command promt, powershell, ataupun git bash
3. Sesuaikan .env dengan database
4. Aktifkan python virtual environment
5. Install "requirements.txt" dengan 'pip install -r requirements.txt'
6. Jalankan server 'uvicorn main:app --reload'
7. Buka 'http://localhost:8000/docs' di browser
